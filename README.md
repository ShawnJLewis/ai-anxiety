# 最后的简历

一个关于 AI 焦虑的多结局文字冒险游戏。

从 ChatGPT 诞生的那一刻开始，你是一个研二学生。你的每一个选择，都在回答一个问题：

**在 AI 时代，我该以什么姿态活下去？**

---

## 试玩

🌐 **在线版**：`https://shawnjlewis.github.io/ai-anxiety/`

📱 **APK**：`https://github.com/ShawnJLewis/ai-anxiety/releases`

---

## 游戏特色

- **多结局**：5 个主要结局 + 隐藏结局，取决于你的选择
- **AI 恋人支线**：和 AI 谈恋爱，《Her》彩蛋有机融合
- **SBTI 人格类型**：每章结束后生成一个荒谬的 AI 焦虑人格类型，可分享到朋友圈
- **数值系统**：焦虑值 / 竞争力 / AI 信任度 / 钱包，影响剧情走向
- **真实时间线**：基于 2022-2025 年真实 AI 发展节点

---

## 游戏结构

```
序章 (2022.11) → 第一章 (2023.3) → 第二章 (2023.9) → 第三章 (2024) → 第四章 (2025)
```

---

## 技术栈

- 单 HTML 文件，纯前端，无需服务器
- 数据与逻辑分离（story.json + index.html）
- 支持手机/PC，支持 GitHub Pages
